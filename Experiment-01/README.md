# Experiment 01

# Data Acquisition, Exploration, and Statistical Analysis

## Aim

To understand the fundamentals of data acquisition, exploratory data analysis (EDA), and statistical analysis using Python libraries such as NumPy and Pandas.

---

## Objectives

- Learn the basics of NumPy and Pandas.
- Load datasets using Pandas.
- Explore datasets using statistical functions.
- Understand descriptive statistics.
- Analyze relationships between variables.

---

## Experiment Tasks

### 📘 Pre-Lab

Answer the theory questions related to EDA, NumPy, Pandas, and statistical measures.

### 💻 In-Lab

- Task-01 : Loading of Dataset and NumPy, Pandas Basics for Data Exploration
- Task-02 : Perform Data Acquisition and Exploratory Analysis
- Task-03 : Analyze Data Distribution using Visualization
- Task-04 : Interpret Statistical Insights

### 📝 Post-Lab

Perform Exploratory Data Analysis on the Titanic Dataset.

---

## Tools Required

- Google Colab
- Python
- NumPy
- Pandas
- Matplotlib
- Scikit-Learn

---

## Outcome

After completing this experiment, students will be able to:

- Load datasets using Pandas.
- Perform exploratory data analysis.
- Calculate statistical measures.
- Visualize datasets.
- Interpret dataset characteristics.
